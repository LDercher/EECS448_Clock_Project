EECS 448 Project #2: You're all FIRED! Do this Clock



A project for EECS 448 - Software Engineering I at the University of Kansas. The continuous implementation of Team 14 (Victor Berger, Jesse Yang, Jeromy Tsai, Cammy Vo)'s project. The original clock is implemented in Java and Swing GUI. It has the functions of setting up the time and changing between standard hour and military hour. We added new features to the clock: Timer, Stop Watch modes and Calendar display.

Our new clock version on Github
Authors: Luke Dercher , Sri Gayatri , Luke Weber , Vuong Nguyen , Dylan Egnoske
Last changed on: March 6th, 2016.
--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

NEW functionalities of the application:
- The users can set up the date on the calendar display.
- Timer: The user will be able to set the time and reset.
- Stop watch: Simple play/pause and reset the time.

Platform and languages: The clock is implemented in Java and its Swing GUI.
